# Artist

Storytelling sites for music artists — built by [Ngoma Charts](https://ngomacharts.netlify.app).

Each folder is a self-contained cinematic page celebrating an artist or release.

## Structure

```
artist/
└── asake-forgiveness/   ← Asake – Forgiveness (coming soon)
```

## About

Ngoma Charts is Kenya's unified music ranking platform.
These pages go beyond the chart — immersive, story-driven experiences for the music that matters.
